Menu Mode Directive
============
Extends the mat-sidenav functionality adding the rail mode.

* Properties

| Name                                 | Description                                         | Default |
|--------------------------------------|-----------------------------------------------------|---------|
| @Input()<br>rail: boolean            | Applies rail mode                                   | 'over'  |
| @Input()<br>isInitiallyOpen: boolean | Whether the menu is initially open or not via input | false   |
